# Calculator

Calculator app powered by vanilla JavaScript.

This was a great project to wrap up the Foundations course. I may go back and rework somethings at a later point. Definitely plan
on implementing physical keyboard support at some point.

I think some of the more challenging parts of this project was validating and "sanitizing" the mathematical equations. I tested a lot
of edge cases, and at this point, I think we're in pretty good standing. I know data validation is very important for not only security,
but for functionality as well. I may have spent just a bit too much time trying to perfect every little detail, but in general, I think
in its current state, the project is ready to go live. 👍

Also want to flex (no pun intended!) the object mapping for the keypad buttons. Using an object actually made life a whole lot easier-
surely saved me a bunch of time!
